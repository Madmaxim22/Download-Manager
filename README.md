## Описание проекта

Download Manager - это веб-приложение, которое позволяет скачивать PDF-файлы с отслеживанием объема скачанных данных. Проект реализует интерфейс для скачивания файлов средствами JavaScript с возможностью отслеживания общего объема скачанных данных в мегабайтах.

## Особенности

- Скачивание PDF-файлов средствами JavaScript
- Отслеживание общего объема скачанных данных
- Отображение объема в мегабайтах
- Повторное скачивание файла учитывается в общей статистике
- Современная архитектура с использованием ES6 модулей

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Madmaxim22/Download-Manager.git
```

2. Установите зависимости:
```bash
yarn install
```

## Использование

1. Запустите режим разработки:
```bash
yarn start
```

2. Сборка для продакшена:
```bash
yarn run build:prod
```

3. Для проверки и исправления кода:
```bash
yarn run lint
```

## Архитектура

Проект состоит из следующих основных компонентов:

- [`DownloadManager.js`](src/classes/DownloadManager.js:1) - класс, отвечающий за скачивание файлов и подсчет объема скачанных данных
- [`FileDisplay.js`](src/classes/FileDisplay.js:1) - класс, отвечающий за отображение списка файлов и обновление информации о скачанных данных
- [`index.js`](src/index.js:1) - основной файл приложения, объединяющий компоненты

## Функциональность

- Скачивание файлов по клику на кнопку
- Автоматическое обновление счетчика скачанных данных
- Отображение объема в мегабайтах с точностью до 2 знаков после запятой
- Поддержка повторного скачивания файлов
